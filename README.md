# yanxuan
## 吸顶导航
	1. 当滚动条碰到导航的时候，导航就会跑到浏览器的顶部，并且永远吸顶
	2. 注册与登录会及购物车会显示在吸顶导航里
## 轮播图组件
	1. 圆点是否显示功能
	2. 改变透明度、改变位置功能
	3. 一次走单个图片、一次走多个图片功能
	4. 循环轮播、非循环轮播功能
	5. 左右到头自定义事件
